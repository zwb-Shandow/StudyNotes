# Python3基本数据类型

## 标准数据类型

- Number(数字)
  - int(长整型)
  - float
  - bool
  - complex(复数)
- String(字符串)
- List(列表)
- Tuple(元组)
- Set(集合)
- Dictionary(字典)

Python3 的六个标准数据类型中:

- **不可变数据**: Number、String、Tuple
- **可变数据**: List、Set、Dictionary

### 数据类型转换

Python 数据类型转换可以分为两种：

- 隐式类型转换
  - 由低精度向高精度转换，避免精度丢失
- 显示类型转换
  - 通过调用预定义函数实现，如int()，float()，str()等
