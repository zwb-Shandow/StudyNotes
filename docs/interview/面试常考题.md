5. C++空类实例化后占用几个字节？

占用一个字节，空类实例化后，也会有独立的内存地址，相当于占有一字节空间，以实现C++统一的对象模型。

```
class A {
  public:
    char a;
};
class B {
  public:
    char a;
    int b;
};
int main(int argc, char** argv) {
  A a;
  B b;
  return 0;
}
```

以上示例中，a占用一个字节，b占用8个字节(内存对齐)
